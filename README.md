Chrome Capture full size screenshot

Install AutohotkeyV2 https://www.autohotkey.com/v2/

Launch Chrome, go to the website, scroll to the bottom.

Click and run the script.
---
Manual method (instead of the script):

Launch Chrome, go to the website, scroll to the bottom.

Press F12 to open DevTools.

Press Ctrl + Shift + P (a command field will open).

Type Capture full size screenshot and click.

Chrome will save the entire page as a PNG image.

Uruchom Chrome, wejdź na stronę, przewiń do końca.

---
Metoda ręczna (zamiast skryptu):

Uruchom Chrome, wejdź na stronę, przewiń do końca.

Naciśnij F12 aby otworzyć DevTools.

Naciśnij Ctrl + Shift + P (otworzy się pole poleceń).

Wpisz Capture full size screenshot i kliknij.

Chrome zapisze całą stronę jako obraz PNG.
